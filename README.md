# Churn Prediction

The goal of this project is to develop a ml api with FastApi for customer churn and try to monotor it with with Grafana

### The data source:

  https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## To run this project

```
docker-compose up
```
